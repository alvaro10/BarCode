Instalar la app (BarCode) desde Android Studio
	Si la app no logra crear el objeto detector:
	1.	Entrar a setting -> app.
	2.	Buscar Google Play Service realizar Clean Cache de la app.
	3.	Volver a instalar la app  (BarCode) desde Android Studio.

	
NOTA: Para agregar el codigo de BarCode
	1.	File -> Build, Execution, Deployment -> Gradle -> uncheck "Offline work"
	2.	Tools -> Android SDK -> Check "Google play services"
